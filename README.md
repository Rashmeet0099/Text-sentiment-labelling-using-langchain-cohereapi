# Text Classifier using LangChain + Gemini

## Features
- Classifies text based on:
  - Sentiment
  - Language
  - Style
  - Topics
  - Political Tendency

## Tech Stack
- Python
- LangChain
- Google Gemini API

## Setup
1. Clone repo and install requirements
2. Add your Gemini API Key to `.env`
3. Run `main.py`

## Run Example
```bash
python main.py
